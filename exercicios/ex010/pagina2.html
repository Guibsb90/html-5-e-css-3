Exemplo de cores em CSS

Representação por nomes

Podemos usar os nomes, em ingles, para representar as cores. É uma opção ruim.

Representação por codigos Hexadecimais

Toda cor é formada por um conjunto de combinações de quantidades de azul, verde e vermelho. Cada cor tem seu código.

Podemos usar o codigo rgb para declarar as cores

R de red, G de green e B de blue. Usamos assim ( olhe codigo fonte) e , botamos os valores de cada uma das cores para formar uma cor (da pra pegar os valores no gimp)

Podemos usar as caracteristicas das cores

Podemos usar as caracteristicas h de matiz (hil em ingles) S de saturação e L de luminosidade (olha codigo fonte)
