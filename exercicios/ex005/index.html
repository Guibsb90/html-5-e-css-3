Listas

Para criar uma lista use a tag ol e dentro da tag ol use a tag li para listar as diferentes coisas

  1. banana
  2. melao
  3. camisinha

dentro da tag OL podemos criar o parametro type , para dessa forma mudar o tipo de "contagem" da lista

  1. ir academia
  2. comer
  3. dormir

temos também as listas sem numeraçao com a tag UL

  • carne
  • frango
  • porco

e podemos também botar o parametro type para mudar os icones antes dos itens da lista (opções square, circle e disque)

  • carne
  • frango
  • carne de capivara

podemos criar uma lista dentro da outra

  1. cortar o cabelo
  2. sair pra comer
    1. tentar nao ficar com pensamentos ruins
  3. comprar bolo
e também podemos adicionar o parametro start para decidir quando começar a lista (em qual ordem, ex: 2, 3, 5, e,b,d, etc)

Exemplo usando o parametro start

  1. sair de casa
  2. correr 10km
  3. ir para academia
    • cantar
    • nadar
    • dormir

Lista de Definições

podemos fazer listas de definição com a tag DL. para as palavras DT (para as palavras ) e DD (para definição). Exemplo:

carne
carme é proteina animal
frango
frango é uma ave
