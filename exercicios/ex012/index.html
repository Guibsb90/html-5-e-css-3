Peso da fonte

Para alterar o peso da fonte, usa-se font-weight (vide pagina do style.css)

Podemos fazer alterações sobre a fonte (vide style.css).

Propriedade shorthand é "simplificar" o codigo. Ao inves de usar varias variaveis , usa-se apenas font: (e vai botando os parametros). Ex: font: italic bolder 3em 'times new roman', sans-serif;

Existem varias opções de shorthand mas nao pode ser usado em tudo

A ordem do shorthand é sempre a mesma: FONT STYLE, FONT WEIGHT, FONT SIZE E FONT FAMILY.
