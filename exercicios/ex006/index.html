Links para download

para criar um link de download como este é necessario criar uma pasta e botar o arquivo que será usado para baixaar no link dentro da pasta e usa-se a tag A para linkar o arquivo na pasta onde vc vai escolher. Use os parametros download= e escreva o nome do arquivo ( atente-se para não botar os parametros fora dos chapeuzinhos) ai, o ultimo parametro é type="application/o tipo do arquivo(ex: pdf)
