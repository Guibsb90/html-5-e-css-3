Modelos de Caixas

Para definir o tamanho da caixa (do elemento), usamos os parâmetros: height para altura e width para largura.

Outros parametros são: border (linha em volta da caixa) border-radius (arredondar as bordas), padding ( engordar a borda ), margin (margem), outline (contorno ao redor da borda)

Box-level

(a tag div é uma box lvl) sempre vai começar em uma linha abaixo e ocupar o espaço todo da pagina e terminar para que o conteudo começe abaixo da caixa. Outros exemplos de tags que geram caixas box-lvl são:
  • div
  • h1 ao h6
  • p
  • main
  • header
  • nav
  • article
  • aside
  • footer
  • form
  • video
    •

Inline-level

(a tag span é uma inline-level) é uma caixa que pode começar em qualquer espaço do conteudo e quando terminar, o conteudo continua. Outros exemplos de tags que geram caixas inline-level:
  • span
  • a
  • code
  • small
  • strong
  • em
  • sup
  • sub
  • label
  • button
  • input
  • select
