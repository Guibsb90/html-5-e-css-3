vamos adicionar uma imagem com a tag img

para inserir imagens dessa forma, temos que baixar uma imagem pelo google (preferencialmente no formato jpeg) e salva-la na mesma pasta que está o index.html

um bom site pra buscar imagens é o http://www.unsplash.com

o arquivo em PNG é o unico que permite transparência nas imagens

Para adicionar uma favicon (pequeno icone que fica na aba ao lado do sitio visitado, você vai precisar baixar um icone, que geralmente está no formto .ico . Se quiser baixar alguns icones prontos, recomendo o site IconArchive. A tag é link na opção favicon, escrita no HEAD

para botar uma imagem pelo endereço, é só clicar com o botão direito em cima da imagem e copiar o link e colar onde ficaria o nome do arquivo
